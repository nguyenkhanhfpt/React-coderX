## Project cuối môn ReactJs - CoderX

1. Cấu trúc thư mục
   - backend
   - coderX (chứa file JSON database)
   - frontend
2. Import databse
   - đặt tên cho database là coderX
3. Chạy backend
```bash
 cd backend -> npm start
```
Server chạy trên host 3001
```bash
 http://localhost:3001/
```
4. Chạy frontend
```bash
 cd frontend -> npm start
```
5. Đăng nhập để vào trang chủ
```bash
 userName: admin
 passord: 123123
```
